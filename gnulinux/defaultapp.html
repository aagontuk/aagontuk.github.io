How to set default application for a file type in Linux

For example lets say you want to change the default application for pdf files. First you have to find out the MIME type for pdf files. For this use following command:

$ mimetype TheBookOfElements.pdf
TheBookOfElements.pdf: application/pdf

From the output you can see for pdf files MIME type is application/pdf. Now to see a list of installed application for opening MIME type application/pdf run following command:

$ grep application/pdf /usr/share/applications/mimeinfo.cache
application/pdf=calibre-gui.desktop;org.gnome.Evince.desktop;libreoffice-draw.desktop;calibre-ebook-viewer.desktop;inkscape.desktop;

You can see for opening pdf files Calibre, Evince etc are installed in the system. To set Evince as the default application for opening pdf files run following command:

$ xdg-mime default org.gnome.Evince.desktop application/pdf

To test

$ xdg-mime query default application/pdf
org.gnome.Evince.desktop

Or

$ xdg-open TheBookOfElements.pdf
