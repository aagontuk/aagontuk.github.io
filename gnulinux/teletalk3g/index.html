##উবুন্টুতে আপনি বেশ কয়েকটি পদ্ধতিতে টেলিটক ত্রিজি মডেম কানেক্ট করতে পারেন। নিচে তার কয়েকটি পদ্ধতি দেয়া হল। তবে এ পদ্ধতিগুলোর মধ্যে ২ নম্বর পদ্ধতিটি আর্থাত্ Mobile Broadband হিসেবে কানেক্ট করাই সবচেয়ে ভাল এবং সহজও। কেননা ড্রাইভার ইন্সটল করে ব্যবহারের ক্ষেত্রে Network Manager এর সাপোর্ট পাবেন না। Network Manager এর সাপোর্ট না পেলে আপনার কিছু কিছু অসুবিধা হতে পারে যেমন: Ubuntu Software Center ইউস করতে পারবেন না, VPN ইউস করতে অসুবিধা হতে পারে, এছাড়া Wifi Hotspot তৈরি করতে পারবেন না।
##মডেম সুইচিং সমস্যার যে সমাধান দেওয়া হয়েছে সেটা আপাতত শুধু ZTE ( সাদা ) মডেমটির জন্য।
##লেখাগুলো মূলত উবুন্টুর জন্য হলেও অন্য লিনাক্স ডিস্ট্রিবিউশনগুলোতে মডেম কানেক্ট করার নিয়ম প্রায় কাছাকাছিই হবে।

উবুন্টুতে টেলিটক ত্রিজি মডেম কানেক্ট করার পদ্ধতি

  1. টেলিটক মডেমের ড্রাইভার ইন্সটল করে কানেক্ট করুন।
  2. Mobile Broadband হিসেবে কানেক্ট করুন।
