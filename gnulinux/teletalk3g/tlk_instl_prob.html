**উবুন্টুতে টেলিটক থ্রিজি ইন্সটল করা যাচ্ছে না!

প্রথমত যে সমস্যাটা হতে পারে সেটা হল QT Runtime Library এবং wvdial আপনার উবুন্টুতে ইন্সটল করা না থাকা। যেহেতু টেলিটক থ্রিজি ড্রাইভার সফটওয়্যারটি QT Framework ব্যাবহার করে এবং ইন্টারনেট কানেকশনের জন্য wvdial ব্যাবহার করে তাই এটা চালাতে হলে আপনার কম্পিউটারে QT3 অথবা তার উপরের ভার্সন এবং wvdial ইন্সটল করা থাকতে হবে।

অপারেটিং সিস্টেম ৩২ বিট হলে:

প্রধানত দুটি উপায়ে আপনি এগুলো ইন্সটল করতে পারেন। প্রথমত আপনি যে কম্পিউটারে ইন্সটল করতে চান অর্থাৎ আপনার উবুন্টু ইন্সটলড কম্পিউটারে যদি ইন্টারনেট কানেকশন থাকে( অন্য কোন মডেম, অথবা ব্রডব্যান্ড এর মাদ্ধমে) তাহলে যে কোন প্যাকেজ ম্যানেজার দিয়ে, যেমন apt-get, Synaptic, অথবা Ubuntu Software Center। apt-get দিয়ে ইন্সটল করার জন্য Ctrl + Alt + T চেপে আপনার টার্মিনাল ওপেন করুন লিখুন sudo apt-get install qt3-qtconfig। পাসওয়ার্ড চাইলে পাসওয়ার্ড দিন। স্ক্রিনে কিছু না দেখালে ভয় পাবেন না। পাসওয়ার্ড দিয়ে Enter চাপুন দেখবেন ইন্সটলেশন শুরু হবে। wvdial ইন্সটল করার জন্য একই উপায়ে টার্মিনাল ওপেন করুন এবং লিখুন sudo apt-get install wvdial। পাসওয়ার্ড দিয়ে Enter চাপুন। অন্যান্য প্যাকেজ ম্যানেজারের যেমন Synaptic অথবা Ubuntu Software Center এর ক্ষেত্রে প্যাকেজের নাম লিখে সার্চ দিন। প্যাকেজ দুটি ইন্সটল হয়ে গেলে টেলিটক ত্রিজির ড্রাইভার ইন্সটল করুন।

আর যদি আপনার কম্পিউটারে ইন্টারনেট কানেকশন না থাকে তাহলে কি করবেন? তাহলে অন্য এমন একটি কম্পিউটারের সাহায্য নিতে হবে যেটাতে ইন্টারনেট কানেকশন আছে এবং এখান থেকে নিচের প্যাকেজগুলো ডাউনলোড করে সেখান থেকে আপনার উবুন্টু ইন্সটল করা কম্পিউটার ট্রান্সফার করে ক্রমানুসারে ম্যানুয়ালি ইন্সটল করতে হবে। ক্রম ঠিক না রাখলে কিন্তু ঝামেলায় পরবেন! প্রথমে নিচের ফাইলগুলো ডাউনলোড (যে কম্পিউটারে ইন্টারনেট কানেকশন আছে সে কম্পিউটারের মাদ্ধমে) করে নিন। যেহেতু আপনার কম্পিউটার ৩২ বিট i386 ক্লিক করুন।

২.libxplc0.3.13
৩.libwvstreams4.4-base
৪.libwvstreams4.4-extras
৫.libuniconf4.4
৬.wvdial
এছাড়া ৩২ বিটের জন্য সবগুলো ফাইল একসাথে ডাউনলোড করে নিতে পারেন এখান থেকে।

ডাউনলোড করা শেষ হলে এবার ইন্সটল করার পালা। ইন্সটল করার জন্য প্রথমে টার্মিনাল দিয়ে যে ফোল্ডারে ফাইল গুলো সেভ করেছেন সে ফোল্ডারে যান। ধরুন আপনি ফাইলগুলো Desktope এ Teletalk_3G নামে একটি ফোল্ডারে সেভ করেছে। তাহলে সেখানে যাওয়ার জন্য টার্মিনালে লিখুন cd /home/username/Desktop/Teletalk_3G। "username" এর জায়গায় উবুন্টু ইন্সটল করার সময় আপনি যে Username বেছে নিয়েছেন সেটা ব্যাবহার করুন। ফোল্ডারে যাওয়ার পর এবার একটি একটি করে ফাইল ইন্সটল করতে হবে। ইন্সটল করার জন্য টার্মিনালে টাইপ করুন sudo dpkg -i "filename"। ধরুন প্রথমে আমাদের libqt3-mt ইন্সটল করতে হবে। ফাইলের নাম সাধারণত একটু বড় হয় এবং এক্সটেনশন যুক্ত থাকে যেমন libqt3-mt.deb। "filename" এর স্থানে আপনাকে পুরোটাই লিখতে হবে। এক্ষেত্রে sudo dpkg -i libqt3-mt.deb। ফাইলের নাম দেখার জন্য টার্মিনালে লিখুন ls, তাহলে আপনি যে ফোল্ডারে আছেন সে ফোল্ডারের সব গুলো ফাইলের তালিকা দেখাবে। এভাবে বাকি ফাইল গুলো ইন্সটল করুন। তবে মনে রাখবেন উপরে যে ক্রম দেয়া আছে সেটা অনুসরণ করতে হবে।

সবগুলো ফাইল ঠিকভাবে ইন্সটল হয়ে গেলে এবার টেলিটক ত্রিজির ড্রাইভার ইন্সটল করুন
