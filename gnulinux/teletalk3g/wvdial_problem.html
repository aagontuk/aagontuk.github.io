ড্রাইভার ইন্সটল করে মডেম ব্যবহার করার ক্ষেত্রে Network Manager ইন্টারনেট কানেকশন ডিটেক্ট করতে না পারার কারন:

সাধারণত মডেম দিয়ে ইন্টারনেট কানেক্ট করার পর কানেকশন Network Manager এ দেখানো হয়। অনেকটা এরকম(লাল দাগ দেয়া) -


কিন্তু যারা ড্রাইভার ইন্সটল করে মডেম কানেক্ট করে তাদের ক্ষেত্রে দেখা যায় ড্রাইভার দিয়ে ইন্টারনের কানেক্ট করার পরও Network Manager ইন্টারনেট কানেকশন পায় না। Network Manager Applet এ এরকম দেখায়(লাল দাগ দেয়া) -

যেকারনে অনেক Application যেমন Ubuntu Software Center, Network Manager এর বিভিন্ন সুবিধা যেমন VPN ব্যবহার করা যায় না।
এ সমস্যার কারণ হচ্ছে ইন্টারনেট কানেক্ট করতে টেলিটক ত্রিজি মডেমের ড্রাইভারের wvdial ব্যবহার করা এবং wvdial বাগযুক্ত হওয়ায়। Launchpad এ এ সম্পর্কিত একটি বাগ রিপোর্ট করা আছে।
