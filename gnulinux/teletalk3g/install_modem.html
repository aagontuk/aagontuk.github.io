##উবুন্টু তে টেলিটক ত্রিজি মডেম চালানোর জন্য ড্রাইভার ইন্সটল করার পদ্ধতিঃ

উবুন্টুতে টেলিটক ত্রিজি কানেক্ট করার জন্য আপনাকে যা করতে হবে তা হলও প্রথমে আপনার মডেমের জন্য ড্রাইভার সফটওয়্যার ইন্সটল করতে হবে। সেটা করার জন্য প্রথমে আপনার মডেমটিকে ইউএসবি পোর্টে ইনসার্ট করুন। ইনসার্ট করার পর দেখবেন আপনার Home Folder এর Computer এর উপরে Teletalk 3G নামে একটি রিমুভেবল মিডিয়া দেখাচ্ছে। সেটা ওপেন করুন। ওপেন করার পর সেখানে তিনটি ফোল্ডার দেখতে পাবেন। একটিতে উইন্ডোসের জন্য ড্রাইভার দেয়া, অন্যটিতে ম্যাক ওএস এর জন্য ড্রাইভার দেয়া এবং একটি লিনাক্স অর্থাৎ আপনার উবুন্টুর ড্রাইভার। তাহলে লিনাক্স নামের ফোল্ডারটি ওপেন করুন। সেখানে PCL_BGTLK.tar.gz নামে একটি কমপ্রেসড টরবল ফাইল দেখতে পাবেন। সেটি ডাবল ক্লিক করে ওপেন করুন। তাহলে দেখবেন এটি Archive Manager দিয়ে ওপেন হবে। এবার এটি আপনার পছন্দের মত যায়গায় Extract করুন। ধরুন এটা আপনি Desktope এ Extract করেছেন।

ঠিক ভাবে Extract সম্পন্ন হলে এবার Terminal ওপেন করুন। Terminal ওপেন করার জন্য আপনার Window কি চাপুন এবং লিখুন Terminal অথবা Ctrl + Alt + T চাপুন তাহলে Terminal পেয়ে যাবেন। এবার Terminal এ লিখুন “cd /আপনার পছন্দের জায়গা যেখানে আপনি ঐ ফাইলটি Extract করেছে/ফাইলটির নাম(PCL_BGTLK)” লিখে Enter চাপুন। যদি Desktop এ করে থাকেন তাহলে লিখুন ” cd /home/username/Desktop/PCL_BGTLK ” লিখে Enter চাপুন। এটা হয়ে গেলে এবার নিচের কোড লিখুন ” sudo ./install.sh ” লিখে Enter চাপুন দেখবেন আপনার টেলিটক ত্রিজি মডেমের ড্রাইভার ইন্সটল হয়ে গেছে।

ইন্সটল হয় নি!!

সুবিধা:

এ ক্ষেত্রে আপনার যে সুবিধা হবে সেটা হল গ্রাফিক্যাল ইউসার ইন্টারফেস সহ একটি সফ্টওয্যার পাবেন। যদি উইন্ডোসে ব্যাবহার করে থাকেন তবে একদম সেরকম। এই সফ্টওয়্যার দিয়ে মডেম কানেক্ট করতে পারবেন। এ ছাড়াও মেসেজ পড়া, পাঠান, কল করা ইত্যাদি করতে পারবেন। এছাড়া Modem Settings পাবেন।

অসুবিধা
